# MHChat (Mental Health Chatbot) - Week 1

This repository is a Django project for a mental-health chatbot demo used as a college project.

## Quick start (Week 1)
1. Create venv: `python -m venv venv`
2. Activate venv (PowerShell): `.\venv\Scripts\Activate.ps1`
3. Install requirements: `python -m pip install django djangorestframework`
4. Run migrations: `python manage.py migrate`
5. Create superuser: `python manage.py createsuperuser`
6. Run server: `python manage.py runserver`
